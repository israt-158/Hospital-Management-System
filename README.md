# Hospital Management System

## Overview
Hospital Management System is a MySQL-based database project designed to manage hospital operations efficiently.

## Features

- Patient Management
- Doctor Management
- Nurse Management
- Appointment Management
- Room & Ward Management
- Medicine Inventory
- Pharmacy Module
- Laboratory Module
- Billing & Payment
- Emergency Patient Management
- Staff Management
- Login System
- Reports & Statistics

## Database Tables

- Users
- Patients
- Doctors
- Nurses
- Staff
- Rooms
- Appointments
- EmergencyPatients
- Medicines
- PharmacySales
- LaboratoryTests
- Bills
- Payments
- Reports
- LoginHistory

## Technologies Used

- MySQL
- SQL
- Database Management System (DBMS)

## Concepts Implemented

- Primary Keys
- Foreign Keys
- CRUD Operations
- Data Validation
- Database Relationships
- Database Normalization
